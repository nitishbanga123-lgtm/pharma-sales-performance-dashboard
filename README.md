# Pharma Sales Performance Dashboard

## Overview
Built a pharma sales analytics solution using Excel, SQL, and Power BI to evaluate product performance, regional sales trends, sales rep effectiveness, and target achievement.

## Tools Used
- SQL
- Power BI
- Microsoft Excel
- KPI Reporting
- Dashboard Design

## Key Features
- Revenue by Product
- Regional Units Sold Analysis
- Sales Representative Performance
- Monthly Revenue Trend
- Visit Type Distribution
- Target Achievement KPI

- Interactive slicers for business filtering

## Business Outcome
Simulated a real-world pharma commercial intelligence dashboard to support strategic decision-making and sales performance evaluation.
